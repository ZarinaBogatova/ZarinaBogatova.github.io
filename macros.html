Макросы

Макрос - это записанная последовательность команд, которую выполняет excel. Вот мои макросы:


Первый макрос - макрос копирования. Он вставляет заданный текст в выделенную ячейку

Sub Макрос1()
Макрос1 Макрос
Сочетание клавиш: Ctrl+й
ActiveCell.FormulaR1C1 = "фывапр"
Range("C5").Select
End Sub

Второй макрос - макрос замены.Он меняет данные документа на заданные данные.

Макрос2 Макрос
Сочетание клавиш: Ctrl+л
Cells.Replace What:="фыв", Replacement:="123", LookAt:=xlPart, _ SearchOrder:=xlByRows, MatchCase:=False, SearchFormat:=False, _ ReplaceFormat:=False
End Sub

Третий Макрос - макрос изменения шрифта. В моем случае он делает курсив и жирными буквы (и все данные) ячейки.Так же она ограничен скриптом на листе с именем "лист 1".

Sub Макрос 3()
сочтание клавиш:Ctrl+н
If Worksheets(ActiveSheet.Name).Name = "лист1" Then
Selection.Font.Bold = True
Selection.Font.Italic = True
End If
End Sub
